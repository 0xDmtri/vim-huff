## Features
This plugin adds [Huff](https://docs.huff.sh) language support for Vim, with the following main features:

- [x] Compile with `:HuffCompile`
- [x] Syntax highlighting
- [x] Lookup evm opcode with 'K'
- [ ] Folding
- [ ] Debug huff program
